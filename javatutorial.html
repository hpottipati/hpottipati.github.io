Introduction to Java
  • Java is an object oriented programming language based off of C and C++. Released in 1995, it has grown to huge popularity, and is now the most used programming language in the world
  • Just like other OOP languages, Java also has implementations of Encapsulation, Inheritance, and Polymorphism
  • The class name should match up to the name of the file that you are running your code is in
  • All the code on this page is able to be run in any IDE or any java-enabled command prompt. Please try these by yourself to really understand them
  • Click here to read an article on how to setup java and the command prompt to turn java programs

Variables
  • Variables are places where data is stored
  • There are many types of Variables, like int, double, boolean, and String. All of these store a different type of data. Let's take a look at a program that shows variables in action
  
                class intVariables{      
                    public static void main(String args[]){
                        int num; //Create an integer variable named num
                        num = 10; //Assign num to a value of 10                
                        System.out.println("Num is: " + num); // Print out Num is: 10, this is because we assigned num to 10 in the previous line
                        //The way to combine things when they are being printed is to put a plus sign 
                        num = num * 2; //set the value of num to twice itself. 10 * 2 = 20. Num is now equal to 20. 
                        System.out.println("Num is now: " + num); //Print out Num is now: 20
                    } 
                }

Result:
Num is: 10
Num is now: 20

Another way to create variables is type name = value;

In the previous code, we learned about integer variables, now let's learn about String variables

Strings are lines of text and represented by double quotes around them. In fact, in the first Hello World program, "Hello World" is a string!

 
                class stringVariables{ 
                    public static void main(String args[]){
                        String x = "Hi"; // Create String variable named x that stores "Hi"
                        String y = "There";// Create String variable named x that stores "There". 
                        System.out.println(x + y);//Prints out x and y ("Hi" and "There") combined. 
                    } 
                }
The Result will be: HiThere. This happens because there is no space between the Strings. While in the println line, there is space between x and y, this space doesn't mean anything because there isn't a string signifying a space.
System.out.println(x + "" + y); adds a space between the two strings and therefore the result will be: Hi There.
Control Statements

Control Statements control where a code run based on if a condition is true or false This is done with the if, else if, and else Statements. The condition is done with relational operators. These compare two values.

 
                class ifControl{ 
                    public static void main(String args[]){
                int x = 1; // initializes int named x and sets it to 1.
                int y = 2; // initializes int named y and sets it to 2.
                if(y > x) System.out.println("y is greater than x"); // Checks if y is greater than x. It is so it prints out y is greater than x.            
                    } 
                }

Let's change this up a little bit by using the else statement. The code under else, runs when all the other conditions are not true.


  
                class ifAndElseControl{ 
                    public static void main(String args[]){
                int x = 1;// initializes int named x and sets it to 1.
                int y = 2;// initializes int named y and sets it to 2.
                if(x > y) System.out.println("y is greater than x"); // Checks if y is greater than x. It is not, so the code doesn't run. However, there is an else statement.   
                else System.out.println("The other conditions are false");// Since the first condition isn't true,  this else block runs and prints out The other conditions are false. 
                    } 
                }

We can use more than one condition using the else if statement

  
                class ifAndElseIfAndElseControl{ 
                    public static void main(String args[]){
                int x = 1; // initializes int named x and sets it to 1.
                int y = 1; // initializes int named y and sets it to 1.
                if(x > y) System.out.println("y is greater than x"); // Checks if y is greater than x. It is not, so the code doesn't run. However, there is an else statement    
                         
                else if(x == y) System.out.println("x is equal to y"); // Checks if x is equal to y and it is so it prints out: x is equal to y.

                else System.out.println("The other conditions are false"); // Since the above else if condition is true, this statement doesn't run. 
                    } 
                }
Blocks of code
  • Curly brackets can be used to group code together. We can use this in our control statements
  • This is useful when we want to run multiple lines of code when a condition is satisfied

                class controlBlocks{ 
                    public static void main(String args[]){
                        int x = 0; //Creates and set an int named x to 0;
                        int y = 1; //Creates and set an int named y to 1;
                        if(x < y){ //Creates a condition thats if x is less than y. It is so this block runs.
                            System.out.println("x is less than y");//Prints out: x is less than y.
                            System.out.println("x is less than 1");//Prints out: x is less than 1.
                        } 
                    }
Loops

A loop statement repeat code multiple times. Let's take a look at one of the types of loops, the for loop

The for loop is written as follows

for(initialization;condition;increment)

You first initialize a variable and set it to value. Then you make a condition for the loop run for. Then you increment (increase) the initialized variable so that it can eventually make the condition not true anymore.

  
                class forLoops{
                    public static void main(String args[]){
                     for(int i = 0; i < 3; i = i + 1){//initializes int named i and sets it equal to 0. This loop runs until i is not less than 3. i increases by 1 every loop. 

                        System.out.println("The number is: " + i);//Prints "The number is: i". i changes every loop because it increments

                } 
            } 
        }

Results:
The number is: 0
The number is: 1
The number is: 2
